# 劳动法合规指南生成技能 — 使用说明

## 这是什么

一个 Claude Code 技能。输入任意国家名，自动生成该国完整的劳动法合规指南 Word 文档，输出到桌面。

面向场景：律所为企业客户批量制作出海目的国劳动法合规报告，或企业法务自行调研海外用工环境。

## 它能做什么

1. 操控浏览器中的 Google NotebookLM，用目标国语言搜索劳动法资料并导入
2. 自动提交 18 个预设问题，提取 NotebookLM 基于导入资料生成的法律分析
3. 通过 DeepSeek 为每个章节生成 150-250 字的专业小结
4. 生成结构完整、排版规范的 Word 文档（封面、前言、法律体系表、六大板块 15 章正文、特别声明）
5. 下载目标国法律 PDF 原文和劳动法律文本模板，建立文件夹结构

整个过程 Claude 自动完成，你只需在开始和结束时各操作一次。

## 工作流程

### 你做的事（5 分钟）

**Step 1 — 粘贴系统提示词。** Claude 会生成一段英文提示词（定义 NotebookLM 的输出风格和结构），你复制粘贴到 NotebookLM 的设置中。

**Step 2 — Discover 搜索导入来源。** Claude 生成 16 个领域的关键词（如"劳动合同订立"、"工作时间与休息休假"），你在 NotebookLM Discover 中逐领域搜索并导入搜索结果。每个领域点几下，16 个领域大约 5-10 分钟。

### Claude 做的事（自动）

**Step 3 — 提交 18 个问题提取答案。** 通过 Chrome DevTools Protocol 操控 NotebookLM 聊天框，逐个提交问题（如"外商投资企业用工主体资格合规"），轮询等待答案生成，提取后保存。

**Step 4 — 生成章节小结和法律体系表。** 调用 DeepSeek API 为 15 个章节各自生成小结；搜索目标国主要劳动法律，收集名称、官方链接和中文简介，建立法律体系表。

**Step 5 — 生成 Word 文档。** 组装封面（目标国名 + 英文副标题 + 律所名称 + 日期）、前言、外商投资环境概述、劳动合规概要、法律体系表、目录、六大板块 15 章正文（每章末附小结）、特别声明、联系人、律所简介。

**Step 6 — 下载 PDF 并打包。** 从官方数据库下载各部法律的 PDF 原文，Google 搜索 10 类劳动法律文本模板 PDF（劳动合同、保密协议、解雇通知等），全部放入输出文件夹。

桌面生成一个文件夹，包含：


## 指南内容结构

固定的 6 部分 15 章，覆盖劳动合规全生命周期：

| 部分 | 章节 |
|------|------|
| 第一部分 用工主体与用工模式合规 | 第1章 外商投资企业用工主体资格合规 |
| | 第2章 用工形式选择 |
| 第二部分 劳动合同全生命周期管理 | 第3章 劳动合同订立合规 |
| | 第4章 劳动合同履行与变更合规 |
| | 第5章 劳动合同解除与终止合规 |
| 第三部分 日常用工核心要素合规 | 第6章 工作时间与休息休假合规 |
| | 第7章 工作薪酬合规管理 |
| | 第8章 公积金及保险合规 |
| | 第9章 劳动保护与职业安全合规 |
| 第四部分 专项合规管理 | 第10章 商业秘密保护与竞业限制合规 |
| | 第11章 特殊员工保护合规 |
| | 第12章 反歧视与反性骚扰合规 |
| 第五部分 企业民主管理与劳资协同 | 第13章 工会运作合规 |
| | 第14章 企业内部民主管理合规 |
| 第六部分 劳动争议解决 | 第15章 劳动争议解决机制 |

每个章节内部以「知识点」为单位组织：先陈述法律内容/实际情况，再给出对中国出海企业的合规建议。

## 使用前准备（仅首次）

### 1. 装依赖

```bash
pip install python-docx pyyaml
```

### 2. 设 DeepSeek API Key（可选）

```bash
export DEEPSEEK_API_KEY="sk-xxxxxxxx"
```

不设也能用，只是章节小结会显示为「待生成」。

### 3. 启动 Chrome 远程调试

macOS:
```bash
/Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome --remote-debugging-port=9222 &
```

### 4. 启动 CDP Proxy

```bash
node scripts/cdp-proxy.mjs
```

保持终端开着。

### 5. 检查环境

```bash
node scripts/check-deps.mjs
```

输出 `proxy: ready` 即可。

### 6. 首次配置

```bash
python3 scripts/setup.py
```

填写律所名称、联系人等信息，生成 config.yaml。后续每个指南的封面和文末都会嵌入这些信息。

### 7. 登录 NotebookLM

Chrome 打开 https://notebooklm.google.com/ 并登录。

## 使用

在 Claude Code 中：

```
/labor-law-guide 韩国
```

然后按 Claude 提示操作即可。

## 注意事项

- 法律内容来自 NotebookLM 基于导入资料的 AI 生成结果，不构成正式法律意见
- NotebookLM 最多支持 300 个资料，本技能 16 个领域约需 100-150 个，在安全范围
- 翻译（法律全文、模板）不在本技能范围内，用户自行处理
- 具体法律问题请咨询目标国执业律师

## 技术架构

```
Claude Code (SKILL.md)
  ├── CDP Proxy (localhost:3456) ←→ Chrome (--remote-debugging-port=9222)
  │     └── NotebookLM (Discover + Chat)
  ├── DeepSeek API (章节小结)
  └── python-docx (Word 文档生成)
```

不依赖 WebSearch/WebFetch，所有法律信息获取在 NotebookLM 浏览器内完成。
