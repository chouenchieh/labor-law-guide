# Labor Law Guide Skill

An automated country-specific labor law compliance guide generator for Chinese enterprises investing abroad. Input a country name, get a complete Word compliance guide on your desktop.

## How It Works

Controls Google NotebookLM via CDP (Chrome DevTools Protocol) Proxy:
Discover searches for labor law sources → submits chapter questions and extracts answers → generates a structured Word document.

```
Country name → NotebookLM Discover sources → 18 Q&A extractions → Word guide
```

## Prerequisites

| Dependency | Version | Notes |
|------------|---------|-------|
| Node.js | 22+ | Native WebSocket support |
| Python | 3.9+ | Document generation scripts |
| Chrome | Any | Must launch with remote debugging enabled |
| Google account | - | Log into NotebookLM |

### 1. Install Python dependencies

```bash
pip install python-docx pyyaml
```

### 2. Set DeepSeek API Key (optional)

Required for chapter summaries. The main guide can be generated without it (summaries will show as pending).

```bash
export DEEPSEEK_API_KEY="sk-xxxxxxxx"
```

### 3. Launch Chrome with remote debugging

**macOS:**
```bash
/Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome --remote-debugging-port=9222 &
```

**Linux:**
```bash
google-chrome --remote-debugging-port=9222 &
```

**Windows:**
```cmd
"C:\Program Files\Google\Chrome\Application\chrome.exe" --remote-debugging-port=9222
```

> Chrome 148+ requires a non-default profile directory. If you encounter issues, create a temporary profile:
> ```bash
> cp -al "$HOME/Library/Application Support/Google/Chrome" /tmp/chrome-profile-link
> /Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome --remote-debugging-port=9222 --user-data-dir=/tmp/chrome-profile-link &
> ```

### 4. Start CDP Proxy

```bash
node scripts/cdp-proxy.mjs
```

Keep this terminal running. Proxy listens on `http://localhost:3456`.

### 5. Verify environment

```bash
node scripts/check-deps.mjs
```

You should see:
```
node: ok (v22.x)
chrome: ok (port 9222)
proxy: ready
```

### 6. Log into NotebookLM

Visit https://notebooklm.google.com/ in Chrome and log into your Google account.

## Usage

Load this skill in Claude Code, then type a country name:

```
/labor-law-guide South Korea
```

Full workflow (see [SKILL.md](SKILL.md)):

| Step | Who | What |
|------|-----|------|
| 1 | You | Paste the system prompt generated by Claude into NotebookLM |
| 2 | You | Use the 16 search keywords generated by Claude to search and import sources via Discover |
| 3 | Claude | Submits 18 questions via CDP and extracts answers from NotebookLM |
| 4 | Claude | Generates chapter summaries via DeepSeek and builds the legal system table |
| 5 | Claude | Generates the Word document on your desktop |
| 6 | Claude | Downloads legal PDFs and organizes the folder structure |

## Disclaimer

- All legal content is generated by NotebookLM based on imported sources and does not constitute formal legal advice
- NotebookLM supports up to 300 sources; 16 areas require approximately 100-150, well within limits
- Consult qualified legal professionals licensed in the target jurisdiction for specific legal matters
