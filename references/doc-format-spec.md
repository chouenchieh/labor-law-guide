# Word 文档格式规范

基于荷兰/日本/加拿大参考文档提取，跨国家统一适用。

## 页面设置

- A4 (210mm × 297mm)
- 四边距：2.54cm

## 字体体系

| 用途 | 字体 | 大小 | 颜色 |
|------|------|------|------|
| 封面中文标题 | 华文隶书 | 36pt | 黑色 |
| 封面英文标题 | Times New Roman | 22pt | 黑色, Bold |
| 封面律所/日期 | 等线/华文隶书 | 16pt | 黑色 |
| 部分标题 (第X部分) | 黑体 | 22pt | #124166 |
| 章节标题 (第X章) | 黑体 | 16pt | 黑色 |
| 正文 | 等线 | 11pt | 黑色, Justified |
| 子标题 (一、二、) | 黑体 | 11pt | 黑色 |
| 法律体系表头 | 黑体 | 11pt | 白色 (#FFF), 底色 #124166 |
| 法律体系表体 | 等线 | 9pt | 黑色 |
| 法律名称超链接 | 等线 | 9pt | #0563C1 (蓝色), 下划线 |
| 目录 | 等线 | 11/10.5pt | 黑色 |

## 法律体系表格

- 双列：法律名称 (5.5cm) | 主要内容 (11cm)
- Table Grid 样式，居中
- 法律名称如有官方 URL，名称本身设为可点击超链接，不另写"官方链接："文本
- 表头行深蓝底色白字

## 文档结构

```
封面 (无页眉)
  ├─ 中文标题 + 英文副标题
  ├─ 律所名称
  └─ 日期
前言 (3段)
  ├─ 段1: 经贸背景 + 投资吸引力 + 劳动法体系特点
  ├─ 段2: 指南编写依据 + 视角 + 覆盖议题 + 体例
  └─ 段3: 法律差异/更新说明 + 执业律师咨询建议
外商投资环境 (自然段落)
劳动合规概要 (自然段落)
劳动法律体系 (双列表格，法律名可点)
目录
第一部分 用工主体与用工模式合规
  ├─ 第1章 外商投资企业在{国}用工主体资格合规
  │   ├─ 内容与合规建议
  │   └─ 本章小结
  └─ 第2章 用工形式选择
第二部分 劳动合同全生命周期管理
  ├─ 第3章 劳动合同订立合规
  ├─ 第4章 劳动合同履行与变更合规
  └─ 第5章 劳动合同解除与终止合规
第三部分 日常用工核心要素合规
  ├─ 第6章 工作时间与休息休假合规
  ├─ 第7章 工作薪酬合规管理
  ├─ 第8章 公积金及保险合规
  └─ 第9章 劳动保护与职业安全合规
第四部分 专项合规管理
  ├─ 第10章 商业秘密保护与竞业限制合规
  ├─ 第11章 特殊员工保护合规
  └─ 第12章 反歧视与反性骚扰合规
第五部分 企业民主管理与劳资协同合规
  ├─ 第13章 {国}工会运作合规
  └─ 第14章 企业内部民主管理合规
第六部分 劳动争议解决全流程应对
  └─ 第15章 {国}劳动争议解决机制
```

## 章节内容格式

每章从 NotebookLM 答案提取，格式为 Q&A 结构：
- `一、法律内容：...` → 子标题（黑体 11pt）+ 正文段落
- `二、合规建议：...` → 子标题（黑体 11pt）+ 正文段落
- 连续文本段落 → 等线 11pt Justified

## 提示框 (Tip)

- 浅灰底色 (#F2F2F2)
- 左侧蓝色边框 (#124166, 12px)
- 等线 11pt

## 超链接

法律体系表中法律名称本身为可点击超链接：
- 蓝色字体 (#0563C1)
- 下划线
- 点击后浏览器打开官方法律全文页面
